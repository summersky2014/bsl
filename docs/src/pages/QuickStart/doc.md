# 介绍

## 特性和优势

* UI 样式高度可配置，拓展性更强，轻松适应各类产品风格；
* 组件丰富、能全面覆盖各类场景；
* 内置各种优化兼容方案，一体式开发；
* 使用<code>TypeScript</code>开发，提供类型定义文件，支持类型及属性智能提示，方便业务开发；

## 适用场景

* 适合于基于<code>react</code>应用
* 适合不同 UI 风格的高度定制需求的应用

## 支持环境

* <code>IOS >= 7</code>
* <code>Android >= 4.4</code>

## 安装

<code>npm i bsl</code>
